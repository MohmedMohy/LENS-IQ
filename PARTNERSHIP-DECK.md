### الـ Endpoints الخاصة بالشركاء

| الـ Endpoint | الوصف | المصادقة |
|-------------|-------|---------|
| `POST /api/v1/partners/evaluate` | تقييم طلب تمويل | API Key |
| `GET /api/v1/partners/applications` | قائمة الطلبات المرسلة | JWT |
| `GET /api/v1/partners/applications/:id` | تفاصيل طلب معين | JWT |
| `POST /api/v1/partners/decision/:id` | إرسال قرار البنك | API Key |
| `GET /api/v1/partners/stats` | إحصائيات الأداء | JWT |
| `POST /api/v1/partners/programs` | إدارة برامج التمويل | JWT |

## 5 | الأمان والامتثال

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  Lens IQ يطبق أعلى معايير الأمان لحماية بيانات العملاء والشركاء    │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  تشفير البيانات                                            │    │
│  │  • AES-256-GCM — تشفير البيانات الحساسة (الرقم القومي،     │    │
│  │    الحسابات البنكية، الهاتف) في قاعدة البيانات              │    │
│  │  • TLS 1.3 — كل الاتصالات بين الطرفين مشفرة                │    │
│  │  • Hashing SHA-256 — لكلمات السر ومفاتيح API                │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  عزل البيانات (Multi-Tenant Isolation)                       │    │
│  │  • كل معرض له قاعدة بيانات مستقلة ومعزولة                   │    │
│  │  • لا يمكن لأي معرض الوصول لبيانات معرض آخر                 │    │
│  │  • كل طلب API يمر بفحص صلاحيات صار                         │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  المصادقة والتحكم بالوصول                                  │    │
│  │  • JWT — توكنات وصول محدودة المدة (15 دقيقة)               │    │
│  │  • Refresh Tokens — دوارة ومؤمنة                             │    │
│  │  • API Keys — لكل شريك مفتاح فريد                             │    │
│  │  • Rate Limiting — حماية ضد سوء الاستخدام                   │    │
│  │  • CORS — نطاقات مصرح بها فقط                               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  السجلات والتدقيق (Audit Log)                               │    │
│  │  • كل عملية مسجلة — من قام بها، متى، وماذا                   │    │
│  │  • سجل كامل لجميع التغييرات على الطلبات                     │    │
│  │  • متاح للتدقيق من قبل الشركاء                             │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  الامتثال                                                   │    │
│  │  • متوافق مع قانون حماية البيانات المصري                    │    │
│  │  • متوافق مع معايير البنك المركزي المصري                    │    │
│  │  • اختبارات اختراق منتظمة — تقاريرها متاحة للشركاء          │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
